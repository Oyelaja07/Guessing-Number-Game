# 🎯 Guessing Number Game

A beginner-friendly Python project where the player has **three chances** to guess the correct secret number. If the player guesses correctly within the three attempts, they win. Otherwise, they lose.

---

## 📖 Project Description

This project was created as part of my Python programming practice. It demonstrates the use of loops, conditional statements, user input, and program flow control.

The game gives the player **three opportunities** to guess a predefined secret number. If the correct number is guessed before the attempts run out, the game ends immediately with a winning message. If all three attempts are used without guessing correctly, the game displays a losing message.

---

## 🚀 Features

- 🎮 Simple and interactive guessing game
- 🔢 Three guessing attempts
- ✅ Displays **"You won"** when the correct number is guessed
- ❌ Displays **"You lose"** after three unsuccessful attempts
- ⛔ Stops the game immediately after a correct guess using `break`
- 💻 Beginner-friendly Python code
